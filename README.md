opencog-ros
===========

Dockerfiles for Robot Operating System (ROS), integrated with Blender and
other add-ons in preparation for integrating with OpenCog.

Docker image structure:

    ├─ros-hydro-deps
      ├─ros-hydro-dev
        ├─ros-hydro-openni
          ├─ros-hydro-percept

    ├─ros-indigo-deps
      ├─ros-indigo-dev
        ├─ros-indigo-blender
